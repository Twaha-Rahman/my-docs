# Source tree and build tree directories (both global and local)

`CMAKE_SOURCE_DIR` ---> Top-level Source Directory (global)
`PROJECT_SOURCE_DIR` -> Source Directory of Current Project (local)

`CMAKE_BINARY_DIR` ---> Top-level Build Directory (global)
`PROJECT_BINARY_DIR` -> Current Project Build Directory (local)

`CMAKE_CURRENT_BINARY_DIR` –-> Binary dir of the current `CMakeLists.txt`
`CMAKE_CURRENT_SOURCE_DIR` –-> Source dir of the current `CMakeLists.txt`
